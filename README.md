# Opinionated nvim setup

# Getting Started

To get started:

```bash
mkdir -p ~/.config/nvim
cd ~/.config/nvim
git clone https://github.com/erickthered/nvim-config.git .
```

Lastly restart nvim and the nvim setup should be complete.

# Uninstalling

You should follow the regular Lazy cleanup procedure:

```bash
rm -Rf ~/.config/nvim ~/.local/state/nvim ~/.local/share/nvim
```
